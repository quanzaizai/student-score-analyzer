# Student Score Analyzer

## 项目说明
这是一个使用 Python 编写的成绩分析小项目。

用户输入一串成绩（中间用空格隔开），程序会对这组成绩进行统计分析，并支持循环运行、异常处理以及输入 `q` 退出程序。

## 当前功能
- 计算总分
- 计算最高分
- 计算最低分
- 计算平均分
- 统计 60-79 分人数
- 统计不及格人数
- 统计良好人数
- 统计优秀人数
- 处理空输入
- 处理非法输入
- 支持输入 `q` 退出程序
- 支持循环运行

## 运行方式

```bash
python main.py
```
